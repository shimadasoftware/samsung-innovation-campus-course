Materiales:
  Resistencia -> 330 Ohm (naranja, naranja, cafe)
  1 Led -> larga (anodo +), corta (cátodo -)
  Cable conectado de 5V a positivo
  Cable conectado de GND a negativo
  Cable conectado de 2 a resistencia

Código:
  void setup() {
    pinMode(2, OUTPUT); // declara pin 2 como salida
  }

  void loop() {
    digitalWrite(2, HIGH); //pone un HIGH (5V) en el pin 2
    delay(1000);           // espera de 1000 milisegundos = 1 segundo
    digitalWrite(2, LOW);  // pone un LOW (0V) en el pin 2
    delay(1000);
  }
